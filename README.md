# Mission-to-Mars

## Overview 

### The goal of this project is to extract data from the NASA Science Mars Exploration website. This methodology and tools used in this project are:
- Beautiful Soup
- Splinter
- MongoDB
- Flask
- Google Chrome Developer Tools
- Jupyter Notebook
- VS Code
- Bootstrap 3
- HTML

## Deliverables
